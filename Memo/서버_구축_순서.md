# 서버 구축 순서

- 1\. 서비스에 필요한 패키지 찾기

        ex) httpd, mariadb-server, php, php-mysqlnd, ...

- 2\. 패키지 버전 확인

        # yum info

- 3\. 패키지 설치

        # yum install

- 4\. 설정 및 확인 (시작을 해야 가능한 경우도 존재함)

        # vim /.../.../*.conf

- 5\. 서비스 시작

        # systemctl restart

- 6\. 서비스 확인

        # systemctl status

- 7\. 방화벽 설정

        #firewall-cmd --add-(port/service)

- 8\. 방화벽 설정 확인

        # firewall-cmd --list-all

- 9\. 포트 상태 확인

        # netstat -nltp | grep
