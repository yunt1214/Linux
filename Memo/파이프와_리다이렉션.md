# 파이프와 리다이렉션

## 파이프 ( | )

- 두 개의 프로그램(혹은 명령)을 연결해주는 역할

######

        # cat anaconda-ks.cfg | less

### 필터링 (grep과 함께 사용)

- 찾고자 하는 글자가 포함된 행만 필터링

######

        # rpm -qa | grep vim
        
        # ps -ef | grep ssh

## 리다이렉션

- 리다이렉션 기호를 사용하여 입출력의 방향을 바꿔줌
- 표준 입력 (키보드 입력) - 0
- 표준 출력 (모니터 화면으로 나오게 정상 결과) - 1
- 표준 에러 (모니터 화면으로 나오는 에러메시지) - 2

### 리다이렉션 기호 ( < , > )

- 표준 입출력의 방향을 바꿔주는 기호

######

        >  : 표준 출력값을 표준 입력으로 바꿔줌 (덮어쓰기)
        >> : 표준 출력값을 표준 입력으로 바꿔줌 (이어쓰기)
        <  : 표준 입력값을 표준 출력으로 바꿔줌
